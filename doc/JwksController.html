= JSON Web Key Set (JWKS) Controller

This class manages the jwks_uri API for the Test Stub. The /jwks API is generally called when a JSON Web Token (JWT), signed by the Test Stub's private key, is sent to the authorization server for authentication. The authorization server uses the public key returned by this API call to decode and authenticate the JWT.

def index
  Rails.logger.debug "========= Call to /jwks endpoint =========="

  # Build JWK set from test stub's public key
  jwks = { 
    keys: [{
      kty:  "RSA", 
      n:    UrlSafeBase64.encode64(Application.public_key.n.to_s(2)), 
      e:    UrlSafeBase64.encode64(Application.public_key.e.to_s(2)),
      alg:  "RS256",
      kid:  "rsa1"
    }]
  }

  #jwks = JSON::JWK.new(Application.public_key)
  render json: jwks
end
